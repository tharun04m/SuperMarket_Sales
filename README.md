# Superstore Sales Dashboard

## Project Overview

The Superstore Sales Dashboard is a Business Intelligence project developed using Power BI and MySQL. The dashboard provides insights into sales performance, profitability, product trends, and regional analysis through interactive visualizations and filters.

## Features

* Total Sales KPI
* Total Profit KPI
* Total Orders KPI
* Monthly Sales Trend Analysis
* Top 5 Products by Sales
* Sales Distribution by Category
* Regional Sales Performance Analysis
* Interactive Region Filter

## Technologies Used

* Power BI
* MySQL
* SQL
* Data Visualization
* Business Intelligence

## Dashboard Insights

* Analyzed 5,899+ sales records.
* Identified top-performing products based on sales.
* Compared sales performance across different regions.
* Monitored monthly sales trends.
* Evaluated category-wise sales distribution.

## Database Operations

* Data Import and Cleaning
* SQL Query Execution
* Aggregation and Grouping
* Sales and Profit Analysis
* KPI Calculation

## Key Learnings

* Data Analysis and Reporting
* KPI Design
